# System intro

This export is a **minimal shareable version** of the mmm memory-convergence system.
It is intentionally trimmed so other users can quickly adopt the core method without inheriting private context.

Included:
- `skills/mmm/SKILL.md`
- `skills/skill-creator-enhanced/SKILL.md`
- `AGENTS.md`
- `MEMORY.md`
- `README.md`

Not included:
- user profiles
- daily memory
- conversation transcripts
- private references
- personal operating context

Recommended adoption order:
1. read `README.md`
2. wire `skills/mmm/SKILL.md` as the first-stop convergence path
3. keep `AGENTS.md` as the repo-level posture file
4. keep `MEMORY.md` minimal and index-like
5. use `skill-creator-enhanced` only when convergence has already determined the skill layer is the right destination
