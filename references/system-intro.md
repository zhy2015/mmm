# System intro

This export is a **minimal public version** of the `mmm` memory-convergence system.
It is intentionally trimmed so other users can adopt the core method without inheriting private context.

## Included

- `skills/mmm/SKILL.md`
- `skills/skill-creator-enhanced/SKILL.md`
- `AGENTS.md`
- `MEMORY.md`
- `README.md`
- `references/start-here.md`
- `references/quickstart.md`
- `references/adoption-map.md`
- `references/adoption-pitfalls.md`
- `examples/minimal-adoption.md`

## Not included

- user profiles
- daily memory
- conversation transcripts
- private references
- personal operating context

## Recommended adoption order

1. read `references/start-here.md`
2. read `README.md`
3. wire `skills/mmm/SKILL.md` as the first-stop convergence path
4. keep `AGENTS.md` as the repo-level posture file
5. keep `MEMORY.md` minimal and index-like
6. use `skill-creator-enhanced` only when convergence has already determined the skill layer is the right destination

## Practical note

`SKILL.md` files are primarily for the AI/runtime.
`README.md` is primarily for human readers deciding whether and how to adopt the system.
